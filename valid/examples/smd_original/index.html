smd_original
The original version of the scan-vacuum test for the twelve pipettes of an SMD placer's revolver head. First all pipettes are checked for vacuum, then a scan check is performed on each of them.
How to Use Open the project smd_original and look at the modelling. Then compare with the modified versions smd_concurrent and smd_new.
