Projects
debounce
A simple model of the debouncing circuit.
How to Use
By generating code from the project debounce, it is possible to produce a simple application using VALID's generated code.
This application has been prepared for the different programming languages and target architectures, each found in its own subdirectory. Each directory includes a workspace debounce.dsw for use with Microsoft® Visual Studio 6.0.
cpp-debounce
The plain debouncing circuit in C++, using the simple baseclass DesCode_Automaton. This application does not use multi-threading and should compile and run on any operating system.
cpp-multithreaded
The debouncing circuit written in C++, using the baseclass DesCode_AutomatonThreaded, which uses multi-threading. In this application, the execution of the automaton is not neccessarily suspended by the task which waits for the user's inputs.
cpp-simulator
The same application as in cpp-debounce, but linked with the DesSimulator library. If this application is started, the VALID Simulator will be opened, so it is possible display the states of the model's automata while the application is running.
c-debounce
The plain debouncing circuit in C, in a single-threaded and operating-system independent version.
