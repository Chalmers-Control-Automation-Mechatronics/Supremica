Description A testing procedure for the twelve pipettes of an SMD placer's revolver head.
Features
Demonstrated		 Conflict checking
Projects
smd_concurrent
A modified version of the original model smd_original. The specification scanvacuumtask has been broken into two automata called scantask and vacuumtask, so the tasks of vacuum checking and scan checking can be executed in parallel.
How to Use Open the project smd_concurrent and check for conflicts. The check reveals that the model has a deadlock, which can be fixed by means of a more modular redesign as shown in example smd_new.
See Also smd_original, smd_new
