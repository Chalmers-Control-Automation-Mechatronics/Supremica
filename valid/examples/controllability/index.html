controllability
Description A small workcell consisting of two simple machines connected by a buffer of capacity 1, also known as "small factory".
Features
Demonstrated		 Controllability checking
Projects
TwoMachinesOneBufferNotControllable
An uncontrollable project consisting of the two machines and a buffer specification stating that the buffer may never overflow or underflow.
TwoMachinesOneBufferControllable
A modified version of the uncontrollable project with a fixed buffer specification.
How to Use Open project TwoMachinesOneBufferNotControllable and see that it is not controllable. The error trace obtained from the controllability check shows that the specification cannot avoid buffer overflow, because it allows starting machine 1 when the buffer is full, and afterwards cannot cope with machine 1 finishing. Project TwoMachinesOneBufferControllable shows how this problem can be fixed by modifying the buffer specification, suppressing the starting of machine 1 as long as the buffer is full.
