TwoMachinesOneBufferSupervisor
A project consisting of the two machines, a supervisor, and a buffer specification stating that the buffer may never overflow or underflow.
How to Use Open project TwoMachinesOneBufferSupervisor. The project contains two machines machine1 and machine2, and the buffer specification bufferSpec stating that the buffer may never overflow or underflow. Furthermore, it contains a large supervisor model called supervisor which allegedly ensures this property. In order to see that this is true, select the specification bufferSpec and invoke "Language Inclusion Check" from the "Analysis" menu. The check is successful and shows that the behaviour of the supervised system is contained within the desired behaviour bufferSpec.
For an example of an unsatisfied specification, it is possible to delete the supervisor automaton and perform the check again. This time, the check will fail and produce a counter example, showing that the two machines alone cannot ensure the specification.
